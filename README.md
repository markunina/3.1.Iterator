# Решение задачи 1 (Бесконечная итерация).

**Условие задачи**


Нужно написать класс, расширяющий Iterable<Integer>, по которому мы будем итерироваться. В процессе итерирования мы будем от него получать случайные числа в диапазоне значений. Диапазон задаётся двумя числами - минимальным и максимальным значениями. Передаются они через параметры конструктора.
